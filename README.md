
# WhatsApp Bot

Este es un bot de WhatsApp básico creado con **Node.js** y la librería **whatsapp-web.js**. Responde a varios comandos enviados en un chat de WhatsApp.

## Requisitos

- **Node.js** y **npm** instalados en tu sistema. Si no los tienes, puedes instalarlos desde [aquí](https://nodejs.org/).
- **Termux** (si estás en Android).

## Instalación

1. Clona este repositorio o descarga los archivos.

2. Si usas Termux, navega a la carpeta donde descargaste el bot:
   ```bash
   cd /WHATSAPP-BOT-MD/del/bot
   ```

3. Instala las dependencias necesarias:
   ```bash
   npm install
   ```

4. Ejecuta el bot:
   ```bash
   node index.js
   ```

5. Escanea el código QR generado con la aplicación de WhatsApp en tu teléfono.

## Comandos disponibles

- `.ping` - Responde con "Pong!"
- `.hello` - Responde con "Hola, ¿cómo estás?"
- `.info` - Responde con una breve descripción del bot.
- `.goodbye` - Responde con "Adiós, ¡nos vemos pronto!"
- `.help` - Muestra la lista de comandos disponibles.

## Notas

- El bot creará un directorio llamado `comandos` si no existe, lo que evita errores relacionados con directorios faltantes.
- Si encuentras algún error o necesitas agregar más comandos, no dudes en modificar el archivo `index.js`.
